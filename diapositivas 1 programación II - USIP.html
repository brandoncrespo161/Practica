IMPLEMENTACIÓN

Setters y Getters


class Celular {
  private: 
    int bateria; // Candado cerrado

  public:
    // Setter: El Guardia de Seguridad
    void setBateria(int nivel) {
      if(nivel > 0 && nivel <= 100) {
        bateria = nivel;
      } else {
        cout << "Error!";
      }
    }
};

Prueba el Setter

Intenta asignar un valor. El sistema rechazará números fuera de 0-100.
