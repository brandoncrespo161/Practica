Domina Excel a Nivel Profesional

Este curso te llevará de ser un usuario competente a un experto en Excel, listo para automatizar tareas, modelar escenarios complejos y crear visualizaciones de datos impactantes.

Módulo 1: Macros con VBA: Automatización Profesional

Conceptos Clave

Macro: Una secuencia de comandos e instrucciones que se almacenan para ser ejecutadas de forma automática. Es la solución para cualquier tarea repetitiva.
VBA (Visual Basic for Applications): Es el lenguaje de programación *detrás* de las macros. Mientras la grabadora de macros escribe un código básico, con VBA puedes crear lógica compleja, bucles, condicionales y funciones personalizadas que la grabadora no puede.

Ejemplo 1 (Paso a Paso): Tu Primera Macro Grabada

Objetivo: Automatizar la creación de un encabezado de reporte (fondo azul, letra blanca en negrita, bordes).

  1. Habilitar la pestaña Programador: Ve a Archivo > Opciones > Personalizar cinta de opciones y asegúrate de que la casilla "Programador" (o "Developer") esté marcada.
  2. Iniciar la Grabadora: En la pestaña Programador, haz clic en "Grabar macro". Asigna un nombre (ej. "FormatoEncabezado") y un atajo de teclado (ej. Ctrl+Shift+E).
  3. Ejecutar las Acciones: Selecciona las celdas A1:E1. Ve a Inicio, aplica un color de relleno azul oscuro, color de fuente blanco, activa la negrita y aplica todos los bordes.
  4. Detener la Grabadora: Vuelve a la pestaña Programador y haz clic en "Detener grabación".
  5. Probar la Macro: Borra el formato, selecciona otras celdas y presiona tu atajo (Ctrl+Shift+E). El formato se aplicará instantáneamente.

Ejemplo 2 (Paso a Paso): Creando una Función Personalizada con VBA

Objetivo: Crear una función que no existe en Excel, como `=CALCULAR_IVA(monto)`.

  1. Abrir el Editor de VBA: Presiona Alt + F11. Se abrirá el Editor de Visual Basic.
  2. Insertar un Módulo: En el panel de la izquierda (Proyecto), haz clic derecho sobre tu archivo (ej. "VBAProject (Libro1)") y selecciona Insertar > Módulo.
  3. Escribir el Código: En la ventana blanca del módulo, escribe el siguiente código VBA:

Function CALCULAR_IVA(monto As Double) As Double
    ' Define la tasa de IVA (ej. 21% en España, 16% en México)
    Const TASA_IVA As Double = 0.16
    
    ' Calcula el IVA
    CALCULAR_IVA = monto * TASA_IVA
End Function
  1. Cerrar el Editor: Simplemente cierra la ventana de VBA (Alt + Q).
  2. Usar la Función: En cualquier celda de Excel, ahora puedes escribir =CALCULAR_IVA(100). El resultado será 16. ¡Has expandido la funcionalidad de Excel!
  3. Importante: Para conservar la macro, debes guardar el libro como "Libro de Excel habilitado para macros (.xlsm)".

Ejercicios Prácticos Resueltos (VBA)

Ejercicio 1: Limpiador de Rango

Objetivo: Crear una macro que borre el contenido (pero no el formato) de un rango específico (ej. "A1:D100").

  1. Presiona Alt + F11, inserta un nuevo Módulo.
  2. Escribe el siguiente código. Sub inicia una macro (un procedimiento).
    3. 
Sub LimpiarDatosReporte()
    ' Selecciona la hoja específica
    Worksheets("Hoja1").Range("A1:D100").ClearContents
End Sub
  3. Cierra el editor (Alt + Q).
  4. Ve a Programador > Macros, selecciona "LimpiarDatosReporte" y haz clic en "Ejecutar". El contenido del rango A1:D100 desaparecerá.

Ejercicio 2: Bucle `For...Next` para Resaltar Celdas

Objetivo: Analizar una columna de "Ventas" (ej. Columna B) y colorear en rojo el fondo de cualquier celda con un valor superior a 500.

  1. En un Módulo, escribe este código:
    2. 
Sub ResaltarVentasAltas()
    Dim celda As Range
    
    ' Itera sobre cada celda en el rango B2 hasta B50
    For Each celda In Range("B2:B50")
        ' Asegura que la celda no esté vacía y sea numérica
        If IsNumeric(celda.Value) And Not IsEmpty(celda.Value) Then
            If celda.Value > 500 Then
                ' vbRed es una constante de color (Rojo)
                celda.Interior.Color = vbRed
            Else
                ' Resetea el color si no cumple
                celda.Interior.Color = xlNone
            End If
        End If
    Next celda
End Sub
  2. Ejecuta la macro. Verás cómo las celdas en la columna B que superen 500 se pintan de rojo.

Ejercicio 3: Cuadro de Mensaje (`MsgBox`) Interactivo

Objetivo: Crear una macro que pregunte al usuario si desea ejecutar la limpieza (del Ejercicio 1) y solo proceder si responde "Sí".

  1. En un Módulo, escribe este código:
    2. 
Sub LimpiezaConConfirmacion()
    Dim respuesta As Integer
    
    ' Muestra un cuadro de mensaje con botones Sí/No
    respuesta = MsgBox("¿Está seguro de que desea borrar todos los datos del reporte?", vbYesNo + vbQuestion, "Confirmar Limpieza")
    
    ' vbYes es la constante si el usuario presiona "Sí"
    If respuesta = vbYes Then
        ' Llama a la otra macro que ya creamos
        LimpiarDatosReporte
        MsgBox "Datos limpiados exitosamente.", vbInformation
    Else
        MsgBox "Operación cancelada.", vbExclamation
    End If
End Sub
  2. Ejecuta "LimpiezaConConfirmacion". Aparecerá un cuadro de diálogo profesional pidiendo confirmación antes de actuar.

Ejercicio 4: Copiar Datos entre Hojas

Objetivo: Automatizar la copia de un resumen de "Ventas" (Rango A1:E10 en "Hoja1") a una hoja de "Historial" (pegándolo en la primera celda vacía).

  1. En un Módulo, escribe este código:
    2. 
Sub ArchivarReporte()
    Dim hojaOrigen As Worksheet
    Dim hojaDestino As Worksheet
    Dim proximaFilaVacia As Long
    
    ' Asignar las hojas a variables (buena práctica)
    Set hojaOrigen = Worksheets("Hoja1")
    Set hojaDestino = Worksheets("Historial")
    
    ' Encontrar la próxima fila vacía en la Col A de la hoja "Historial"
    proximaFilaVacia = hojaDestino.Cells(hojaDestino.Rows.Count, "A").End(xlUp).Row + 1
    
    ' Copiar y pegar
    hojaOrigen.Range("A1:E10").Copy
    hojaDestino.Cells(proximaFilaVacia, "A").PasteSpecial xlPasteValues
    
    ' Limpiar el portapapeles
    Application.CutCopyMode = False
    MsgBox "Reporte archivado en fila " & proximaFilaVacia, vbInformation
End Sub
  2. Asegúrate de tener una hoja llamada "Historial". Cada vez que ejecutes esta macro, el reporte de "Hoja1" se copiará al final de la lista en "Historial" (pegando solo valores).

Ejercicio 5: Crear un Botón para Ejecutar Macro

Objetivo: Hacer la macro "LimpiezaConConfirmacion" accesible para usuarios no técnicos mediante un botón en la hoja de cálculo.

  1. Ve a la pestaña Programador > Insertar > Controles de Formulario > Botón.
  2. Haz clic y arrastra en la hoja de Excel para dibujar el botón.
  3. Al soltar el clic, aparecerá una ventana: "Asignar macro".
  4. Selecciona "LimpiezaConConfirmacion" de la lista y haz clic en "Aceptar".
  5. Haz clic derecho en el botón para "Editar texto" y cámbialo a algo descriptivo, como "Limpiar Reporte".
  6. ¡Listo! Ahora cualquier usuario puede hacer clic en ese botón para ejecutar tu macro de forma segura.

Módulo 2: Modelos Financieros y Simulaciones

Conceptos Clave

Modelo Financiero: Una representación matemática de la realidad financiera de una empresa o proyecto. Se usa para tomar decisiones, valorar activos (ej. un Flujo de Caja Descontado - DCF) o analizar el impacto de diferentes variables.
Simulación: Es la práctica de "jugar" con el modelo para responder preguntas de tipo "¿Qué pasaría si...?" (What-If Analysis). Excel ofrece herramientas poderosas para esto.

Herramienta 1 (Paso a Paso): Buscar Objetivo (Goal Seek)

Objetivo: Quieres saber cuántas unidades necesitas vender para alcanzar un punto de equilibrio (ganancia de $0).

  • Planteamiento:
  • Celda B1 (Unidades): 100 (Variable a cambiar)
  • Celda B2 (Precio): $50
  • Celda B3 (Costo Unitario): $30
  • Celda B4 (Costos Fijos): $5,000
  • Celda B5 (Ganancia): =(B1*B2) - (B1*B3) - B4 (Esta es tu celda objetivo)
  1. Ve a la pestaña Datos > Análisis de hipótesis (o Análisis Y si) > Buscar Objetivo.
  2. Definir la celda: B5 (la celda de la Ganancia).
  3. Con el valor: 0 (tu objetivo, el punto de equilibrio).
  4. Para cambiar la celda: B1 (la celda de Unidades).
  5. Haz clic en "Aceptar". Excel iterará y encontrará que necesitas vender 250 unidades para llegar a $0 de ganancia.

Herramienta 2 (Paso a Paso): Administrador de Escenarios

Objetivo: Comparar rápidamente el resultado de tu modelo financiero en un escenario "Optimista", "Pesimista" y "Realista".

  1. Usa el mismo planteamiento anterior. Las celdas que cambiarán son B2 (Precio) y B3 (Costo Unitario).
  2. Ve a Datos > Análisis de hipótesis > Administrador de escenarios.
  3. Haz clic en "Agregar...".
  4. Nombre del escenario: "Pesimista". Celdas cambiantes: $B$2,$B$3.
  5. Introduce los valores pesimistas (ej. Precio: 40, Costo: 35).
  6. Repite el proceso para un escenario "Optimista" (ej. Precio: 60, Costo: 25).
  7. En el Administrador de Escenarios, ahora puedes hacer doble clic en "Pesimista" u "Optimista" para ver cómo cambia tu Ganancia (B5) al instante.
  8. Para un reporte final, haz clic en "Resumen..." y Excel creará una nueva hoja comparando todos los escenarios.

Concepto Avanzado: Simulación de Monte Carlo

La Simulación de Monte Carlo lleva esto un paso más allá. En lugar de 3 escenarios, simula 10,000 escenarios usando números aleatorios.

Ejemplo Conceptual: En lugar de fijar la demanda, la modelas con una función como =ALEATORIO.ENTRE(80, 120) (demanda entre 80 y 120) o =INV.NORM(ALEATORIO(), 100, 10) (distribución normal con media 100 y desv. est. 10). Luego, usas una "Tabla de Datos" o VBA para recalcular tu ganancia miles de veces y analizar la *probabilidad* de éxito.

Ejercicios Prácticos Resueltos (Modelos y Simulación)

Ejercicio 1: Cálculo del VAN (VPN)

Objetivo: Determinar si un proyecto es rentable calculando su Valor Actual Neto (VAN o VPN).

  • Planteamiento:
  • Celda B1 (Tasa Descuento/TMAR): 10%
  • Celda B2 (Inversión Inicial): -100000 (en negativo)
  • Celda B3 (Flujo Año 1): 30000
  • Celda B4 (Flujo Año 2): 40000
  • Celda B5 (Flujo Año 3): 50000
  1. En una celda (ej. B7), vamos a calcular el VAN.
  2. Escribe la fórmula: =VNA(B1, B3:B5) + B2 (En inglés: =NPV(B1, B3:B5) + B2).
  3. Explicación: La función VNA/NPV descuenta los flujos *futuros* (B3:B5) a la tasa B1. La inversión inicial (B2) se suma *fuera* de la función porque ya está en valor presente (Año 0).
  4. Resultado: $1,803.16. Como es positivo, el proyecto es rentable.

Ejercicio 2: Cálculo de la TIR

Objetivo: Encontrar la Tasa Interna de Retorno (TIR) del proyecto anterior (la tasa que hace el VAN = 0).

  1. Usa el mismo planteamiento del Ejercicio 1.
  2. En una celda (ej. B8), escribe la fórmula: =TIR(B2:B5) (En inglés: =IRR(B2:B5)).
  3. Explicación: A diferencia de VNA, la función TIR/IRR SÍ debe incluir la inversión inicial (B2) dentro del rango.
  4. Resultado: 12.59%. Como 12.59% (TIR) es mayor que 10% (Tasa de Descuento), el proyecto se acepta.

Ejercicio 3: Tabla de Datos de 1 Variable

Objetivo: Analizar cómo cambia la "Ganancia" (del modelo de "Buscar Objetivo") si el "Precio" varía de $40 a $60.

  1. Usa el modelo de "Buscar Objetivo": B1=250 (unidades), B2=50 (precio), B3=30 (costo), B4=5000 (fijos), B5=Ganancia (=(B1*B2)-(B1*B3)-B4).
  2. En la celda D2, escribe los precios que quieres probar: 40.
  3. En D3: 45. En D4: 50. En D5: 55. En D6: 60.
  4. En la celda E1 (la celda superior derecha de la tabla), vincula el resultado que quieres observar: =B5.
  5. Selecciona todo el rango de la tabla (D1:E6).
  6. Ve a Datos > Análisis de hipótesis > Tabla de Datos.
  7. En el campo "Celda de entrada (Columna)": selecciona $B$2 (la celda del Precio en tu modelo original).
  8. Haz clic en Aceptar. Excel llenará automáticamente la columna E mostrando la ganancia para cada precio.

Ejercicio 4: Tabla de Datos de 2 Variables

Objetivo: Analizar la Ganancia (B5) variando "Precio" (B2) y "Costo Unitario" (B3) al mismo tiempo.

  1. Usa el mismo modelo (B1 a B5).
  2. En la celda D1 (esquina), vincula el resultado: =B5.
  3. En la columna D (D2, D3, D4...), pon los "Precios" a probar (ej. 40, 50, 60).
  4. En la fila 1 (E1, F1, G1...), pon los "Costos" a probar (ej. 25, 30, 35).
  5. Selecciona todo el rango de la tabla (D1:G4).
  6. Ve a Datos > Análisis de hipótesis > Tabla de Datos.
  7. "Celda de entrada (Fila)": $B$3 (el Costo Unitario, que está en la fila).
  8. "Celda de entrada (Columna)": $B$2 (el Precio, que está en la columna).
  9. Haz clic en Aceptar. Excel creará una matriz mostrando todas las combinaciones de ganancia.

Ejercicio 5: Usando Solver (Optimización)

Objetivo: Maximizar la ganancia de 2 productos (A y B) con restricciones de producción (ej. horas de máquina).

  • Planteamiento:
  • B1 (Unidades Prod A): 10 | C1 (Unidades Prod B): 10 (Celdas a cambiar)
  • B2 (Ganancia/u Prod A): $20 | C2 (Ganancia/u Prod B): $25
  • B3 (Horas/u Prod A): 2 | C3 (Horas/u Prod B): 3
  • B5 (Horas Totales Usadas): =(B1*B3)+(C1*C3)
  • B6 (Horas Disponibles): 100 (Restricción)
  • B8 (Ganancia Total): =(B1*B2)+(C1*C2) (Celda Objetivo)
  1. Ve a Datos > Solver (si no aparece, ve a Archivo > Opciones > Complementos > Complementos de Excel > Ir... y activa "Solver").
  2. Establecer Objetivo: $B$8.
  3. Para: "Max" (Maximizar).
  4. Cambiando las celdas de variables: $B$1:$C$1.
  5. Sujeto a las restricciones (Clic en "Agregar"):
    • $B$5 <= $B$6 (Horas usadas <= Horas disponibles).
    • $B$1:$C$1 >= 0 (No se pueden producir unidades negativas).
    • $B$1:$C$1 = entero (Opcional: si solo puedes vender unidades completas).
  6. Método de resolución: "Simplex LP".
  7. Haz clic en "Resolver". Solver encontrará la combinación óptima (ej. 50 de Prod A, 0 de Prod B) para maximizar la ganancia.

Módulo 3: Dashboards Interactivos y Visualización Avanzada

Conceptos Clave

Dashboard (Cuadro de Mando): Es una visualización de los indicadores de rendimiento clave (KPIs) en una sola pantalla, que permite monitorear la salud de un negocio o proyecto de un vistazo.
Interactivo: Significa que el usuario (ej. un gerente) puede filtrar la información (por año, por región, por producto) sin "romper" el dashboard, y todos los gráficos se actualizan solos.

El Pilar del Dashboard: Tablas y Gráficos Dinámicos

La forma más rápida y robusta de crear dashboards en Excel es usando Tablas Dinámicas (PivotTables) como motor de cálculo y Gráficos Dinámicos (PivotCharts) como visualización.

  1. Prepara tus Datos: Asegúrate de que tus datos estén en un formato tabular (columnas con encabezados, sin filas en blanco) y, preferiblemente, formateados como una Tabla (Insertar > Tabla o Ctrl+T). Esto hace que se actualice automáticamente.
  2. Crea la Tabla Dinámica: Selecciona tus datos. Ve a Insertar > Tabla Dinámica.
  3. Define el KPI: Arrastra "Región" a Filas y "Ventas" a Valores. Ya tienes un resumen de ventas por región.
  4. Crea el Gráfico Dinámico: Con la tabla dinámica seleccionada, ve a Análisis de tabla dinámica > Gráfico Dinámico. Elige un gráfico de barras.
  5. Limpia el Gráfico: Haz clic derecho en los botones grises del gráfico (como "Total" o "Región") y selecciona "Ocultar todos los botones de campo en el gráfico" para un look más limpio.

Añadiendo Interactividad (Paso a Paso): Segmentación de Datos (Slicers)

Los Slicers son filtros visuales e intuitivos.

  1. Haz clic en tu Tabla Dinámica (o Gráfico Dinámico).
  2. Ve a Análisis de tabla dinámica > Insertar segmentación de datos.
  3. Selecciona los campos por los que quieres filtrar (ej. "Año" y "Categoría de Producto").
  4. Aparecerán dos "cajas" con botones (un Slicer para Año, otro para Categoría).
  5. Conectar Múltiples Gráficos: Si tienes varios gráficos (basados en *diferentes* tablas dinámicas), debes conectarles el mismo slicer.
    • Haz clic derecho en el Slicer > Conexiones de informes.
    • Marca las casillas de todas las Tablas Dinámicas que quieres que este slicer controle.
  6. ¡Listo! Ahora, al hacer clic en "2024" o "Electrónica", todos tus gráficos se actualizarán al mismo tiempo.

Visualización Avanzada

No te limites a barras y líneas. Explora Insertar > Gráficos para encontrar:

  • Gráficos de Mapa: Perfecto para datos geográficos (ej. ventas por país o provincia).
  • Gráficos de Embudo (Funnel): Ideal para visualizar procesos de ventas o conversión.
  • Gráficos de Cascada (Waterfall): Muestra cómo un valor inicial es afectado por valores positivos y negativos (ej. Ganancia Neta).

Ejercicios Prácticos Resueltos (Dashboards y Visualización)

Ejercicio 1: Gráfico de Tendencia (Líneas)

Objetivo: Crear un gráfico dinámico que muestre la evolución de las ventas a lo largo del tiempo (meses).

  1. Asegúrate de que tus datos base (Tabla) tengan una columna de "Fecha" (con fechas completas, ej. 01/01/2024).
  2. Crea una nueva Tabla Dinámica (en una hoja separada, ej. "TD_Data").
  3. Arrastra el campo "Fecha" a Filas. Excel automáticamente lo agrupará por Años, Trimestres y Meses.
  4. Arrastra "Ventas" a Valores.
  5. Haz clic en la Tabla Dinámica, ve a Análisis de tabla dinámica > Gráfico Dinámico.
  6. Elige un gráfico de Línea con Marcadores.
  7. Mueve este gráfico (Crtl + X) a tu hoja de "Dashboard" (Crtl + V) y ajústalo.

Ejercicio 2: Insertar una Escala de Tiempo (Timeline)

Objetivo: Añadir un filtro de fechas profesional (Escala de Tiempo) a tu gráfico de tendencia.

  1. Haz clic en el gráfico dinámico de tendencia que acabas de crear.
  2. Ve a Análisis de gráfico dinámico > Insertar escala de tiempo.
  3. Selecciona el campo "Fecha" (solo aparecerán campos de fecha).
  4. Aparecerá un "Slicer" especial que te permite filtrar por Años, Trimestres, Meses o Días, y arrastrar el rango.
  5. (Opcional) Conecta esta escala de tiempo a otros gráficos (clic derecho > Conexiones de informes).

Ejercicio 3: KPI Principal (Tarjeta de "Total Ventas")

Objetivo: Mostrar el número "Total de Ventas" en grande, como un KPI principal.

  1. Crea una nueva Tabla Dinámica (en "TD_Data") que solo tenga "Ventas" en el área de Valores. (Será una sola celda con el gran total).
  2. Nombra esta celda (ej. "TotalVentas").
  3. En tu hoja de "Dashboard", inserta un Cuadro de Texto (Insertar > Texto > Cuadro de Texto).
  4. Con el cuadro de texto seleccionado, NO escribas en él. Ve a la barra de fórmulas.
  5. Escribe = y haz clic en la celda de la tabla dinámica que contiene el gran total (ej. =TD_Data!B4).
  6. Da formato al Cuadro de Texto: quita el borde, quita el relleno, aumenta el tamaño de la fuente (ej. 48pt) y centra el texto.
  7. Importante: Asegúrate de que los Slicers (del Módulo 3, Ejemplo 2) también estén conectados a esta nueva tabla dinámica. Ahora, cuando filtres, ¡tu KPI principal se actualizará!

Ejercicio 4: Formato Condicional en Tablas Dinámicas

Objetivo: Crear una tabla "Top 5 Vendedores" que incluya barras de datos visuales.

  1. Crea una Tabla Dinámica: "Vendedor" en Filas, "Ventas" en Valores.
  2. Ordena de Mayor a Menor (Clic derecho en un valor de ventas > Ordenar > De mayor a menor).
  3. Aplica un filtro de Top 5 (Clic en la flecha de "Etiquetas de fila" > Filtros de valor > 10 mejores... > Cambia 10 a 5).
  4. Selecciona solo las celdas de valores numéricos (las ventas) en la tabla dinámica.
  5. Ve a Inicio > Formato Condicional > Barras de datos > Relleno degradado (Azul).
  6. ¡Ahora tienes un mini-gráfico dentro de tu tabla que se actualiza con los filtros!

Ejercicio 5: Gráfico Combinado (Eje Secundario)

Objetivo: Mostrar "Ventas" (millones) y "% de Crecimiento" (porcentaje) en el mismo gráfico, aunque tengan escalas diferentes.

  1. Crea una Tabla Dinámica: "Mes" en Filas, "Ventas" en Valores.
  2. Añade "Ventas" una *segunda vez* a Valores.
  3. Haz clic derecho en este segundo campo de "Ventas" > Mostrar valores como > % de diferencia de > (Campo base: Mes, Elemento base: (Anterior)). Renómbralo a "% Crecimiento".
  4. Crea un Gráfico Dinámico a partir de esta tabla.
  5. Haz clic derecho en el gráfico > Cambiar tipo de gráfico > Combinado.
  6. Para la serie "Ventas", elige "Columnas Agrupadas".
  7. Para la serie "% Crecimiento", elige "Línea" y marca la casilla "Eje secundario".
  8. Acepta. Ahora tienes un gráfico profesional que muestra la magnitud (barras) y la tendencia (línea) juntas.

Módulo 4: Integración con Otras Herramientas

Concepto Clave

Excel ya no trabaja solo. Es parte de un ecosistema de datos. La habilidad clave es saber cuándo usar Excel y cómo conectarlo con herramientas más potentes (como Power BI) o más colaborativas (como Google Sheets).

Integración con Power BI

Cuándo usarlo: Cuando tu dashboard de Excel se vuelve lento, manejas millones de filas, o necesitas consolidar múltiples fuentes (ej. un Excel, una base de datos SQL y una web) en un solo modelo de datos.

Paso a Paso (La conexión más simple):

  1. En Power BI Desktop (la aplicación gratuita de Windows).
  2. En la pestaña Inicio, haz clic en Obtener datos > Excel.
  3. Selecciona tu archivo .xlsx.
  4. Aparecerá un "Navegador" mostrándote las hojas y las Tablas dentro de tu archivo. (Pro-Tip: Siempre importa Tablas, no hojas).
  5. Selecciona la Tabla y haz clic en "Cargar".
  6. Los datos de tu Excel ahora están en el "Modelo" de Power BI, listos para crear visualizaciones mucho más potentes y rápidas.

Conexión Clave: La herramienta "Obtener y transformar datos" (Power Query) en la pestaña Datos de Excel es *exactamente* el mismo motor que usa Power BI. Si aprendes a limpiar y conectar datos allí, ya sabes el 70% de Power BI.

Integración con Google Sheets

Cuándo usarlo: Cuando la colaboración en tiempo real es más importante que la potencia de cálculo. Múltiples usuarios pueden editar el mismo archivo simultáneamente.

Flujo de Trabajo (Compatibilidad):

  1. Subir un Excel a Sheets: En Google Drive, haz clic en Nuevo > Subir archivo. Sube tu .xlsx. Al abrirlo, puedes elegir "Abrir con Google Sheets".
  2. Descargar de Sheets a Excel: En Google Sheets, ve a Archivo > Descargar > Microsoft Excel (.xlsx).

¡La Trampa de la Integración! (VBA vs. Apps Script)

ADVERTENCIA FUNDAMENTAL: Tus Macros de VBA NO funcionan en Google Sheets.

Si abres un archivo .xlsm en Google Sheets, las macros no se ejecutarán. Google Sheets tiene su propio lenguaje de automatización llamado Google Apps Script (que se parece más a JavaScript).

La Regla:

  • Si la tarea es automatización pesada y análisis complejo en una sola PC: Usa Excel + VBA.
  • Si la tarea es colaboración en vivo y formularios web: Usa Google Sheets + Apps Script.

Ejercicios Prácticos Resueltos (Integración)

Ejercicio 1: (Power Query) Importar Tabla de una Web

Objetivo: Traer a Excel una tabla de datos (ej. población mundial) desde una página de Wikipedia y hacerla actualizable.

  1. En Excel, ve a la pestaña Datos > Obtener datos > De otras fuentes > De la Web.
  2. Pega la URL de una página de Wikipedia que contenga tablas (ej. https://es.wikipedia.org/wiki/Anexo:Países_por_población_(Naciones_Unidas)).
  3. Excel analizará la web y te mostrará un "Navegador" con las tablas que encontró (ej. "Table 0", "Table 1").
  4. Selecciona la tabla que te interesa y haz clic en "Transformar datos".
  5. Se abrirá el editor de Power Query. Aquí puedes limpiar los datos (ej. quitar columnas innecesarias, cambiar tipo de dato).
  6. Cuando estés listo, haz clic en Inicio > Cerrar y cargar.
  7. Los datos de la web ahora están en tu hoja de Excel. Para actualizar: Datos > Actualizar todo.

Ejercicio 2: (Power Query) Consolidar Archivos de una Carpeta

Objetivo: Tienes 12 archivos de Excel (uno por mes) en una carpeta. Quieres unirlos todos en una sola tabla maestra.

  1. Ve a Datos > Obtener datos > De un archivo > De una carpeta.
  2. Navega y selecciona la carpeta que contiene tus 12 archivos.
  3. Aparecerá una vista previa de los archivos. Haz clic en "Combinar y transformar datos".
  4. Power Query te preguntará qué elemento quieres combinar (ej. "Hoja1" de cada archivo).
  5. Automáticamente creará una consulta que abre cada archivo, extrae la "Hoja1" y la "anexa" (une una debajo de otra) en una sola tabla gigante.
  6. Limpia si es necesario (ej. filtrar encabezados repetidos) y haz clic en "Cerrar y cargar".

Ejercicio 3: (Google Sheets) Importar Rango de otra Hoja

Objetivo: En Google Sheets, traer datos en vivo desde otra hoja de cálculo (ej. un "Maestro de Precios" en otro archivo).

  1. Abre tu hoja de Google Sheets de destino.
  2. En una celda, escribe la función =IMPORTRANGE("URL_o_ID_de_la_otra_hoja", "NombreHoja!A1:B10").
  3. URL/ID: Puedes copiar la URL completa de la otra hoja de cálculo.
  4. Rango: El nombre de la pestaña y el rango que quieres importar (ej. "Precios!A1:E50").
  5. La primera vez, mostrará un error #REF!. Pasa el mouse sobre la celda y haz clic en "Permitir acceso" para conectar las dos hojas.
  6. Los datos aparecerán y se actualizarán automáticamente si la hoja de origen cambia.

Ejercicio 4: (Google Forms) Conectar un Formulario a Sheets

Objetivo: Crear un formulario de captura de datos (ej. "Registro de Gastos") que alimente una hoja de cálculo en tiempo real.

  1. Ve a Google Forms (forms.google.com) y crea un formulario simple (ej. "Fecha", "Monto", "Categoría").
  2. En la pestaña "Respuestas" dentro de Google Forms, haz clic en el ícono verde de Google Sheets.
  3. Elige "Crear una nueva hoja de cálculo" y asígnale un nombre.
  4. Se abrirá una nueva hoja de Google Sheets. Verás que las columnas coinciden con tus preguntas del formulario.
  5. Envía una respuesta de prueba desde tu formulario. Verás cómo aparece una nueva fila en tu hoja de Google Sheets instantáneamente.

Ejercicio 5: (Power BI) Analizar Datos de Excel

Objetivo: Recrear un gráfico dinámico simple de Excel dentro de Power BI Desktop.

  1. Guarda tu archivo de Excel con los datos de ventas (el que usaste para los dashboards).
  2. Abre Power BI Desktop.
  3. En Inicio > Obtener datos > Excel. Selecciona tu archivo.
  4. En el Navegador, selecciona la Tabla de datos (no la hoja) y haz clic en "Cargar".
  5. A la derecha, en el panel "Datos", verás tus columnas (Ventas, Región, Fecha...).
  6. En el panel "Visualizaciones", haz clic en el ícono de "Gráfico de barras agrupadas".
  7. Arrastra el campo "Región" al eje Y.
  8. Arrastra el campo "Ventas" al eje X.
  9. ¡Has creado el mismo gráfico que en Excel! Ahora explora cambiar los colores, filtros y otros tipos de visualización en Power BI.
