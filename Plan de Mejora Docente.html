El Arma Secreta: ¡Simulador de Debugging!

El Código Misterioso (con un bug)

num = 1
while num <= 10:
    es_primo = True
    divisor = 2
    while divisor < num:
        if num % divisor == 0:
            es_primo = False
        divisor += 1
    if es_primo:
        print(f"{num} es primo")
    num += 1

¡Es tu turno de ser el detective!

Usa los botones para ejecutar el código y ver qué imprime la consola.

Salida de la Consola (Print)

¡AQUÍ ESTÁ EL ERROR!

La variable `es_primo` nunca se reinicia a `True`. Una vez que se vuelve `False` (para el número 4), la consola deja de imprimir. ¡El bug es visible!

👩‍🏫 Valoración del Docente

La IA es un "asistente de productividad", no un sustituto. Libera tiempo para enfocarse en lo humano: motivar, inspirar y guiar.
